## Projects

**[Gesture AI](https://github.com/Anubhavsingh311/handdetection)** · [Live Demo](https://handdetection-nine.vercel.app/)

**[Fraud Detection Pipeline](https://github.com/Anubhavsingh311/Fraud-Detection-Pipeline)**
An end-to-end fraud-detection workflow from raw transaction data to modeled risk signals, organized as a repeatable Python pipeline from preparation through model evaluation.
`Python` `Pandas` `scikit-learn`

**[NLP & Sentiment Analysis](https://github.com/Anubhavsingh311/NLP_and_Sentiment-Analysis)**

`Python` `NLTK` `TF-IDF` `Naive Bayes`

A retail segmentation study on 1,200 transaction records: 14 raw fields engineered into 20 features, compressed to 13 principal components, and translated into 5 customer personas.
`Python` `PCA` `K-Means` `Seaborn`

**[AudioCast](https://github.com/Anubhavsingh311/AudioCast)** · [Live Demo](https://anubhavsingh311.github.io/AudioCast/)
A privacy-first PDF reader that turns text-based PDFs into an in-browser listening experience. Multi-PDF drag-and-drop, playback speeds from 0.5× to 2×, and live sentence tracking.
